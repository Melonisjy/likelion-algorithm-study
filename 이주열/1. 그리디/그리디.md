# 그리디(Greedy)

그리디 = 탐욕법<br>
여기서 탐욕적이란 `현재 상황에서 지금 좋은 것만 고르는 방법`을 의미한다.<br>
또한 그리디 문제에서 `가장 큰 순서대로` 혹은, `가장 작은 순서대로`와 같은 키워드를 통해 힌트를 얻게 된다.<br><br>

--- 

## 예제 3-1
```
a = int(input())
count = 0

list = [500, 100, 50, 10]
for coin in list:
    count += a // coin # list의 인덱스 0인 500부터 a를 나눈 몫을 count에 + 함.
    a %= coin # a를 a와 coin으로 나눈 나머지로 초기화

print(count)
```
